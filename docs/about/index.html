Research Theme

Four-Dimensional Navigation of Drones Suitable for Unmanned Traffic Management


Today, drones are rapidly becoming a part of our lives due to technological developments, and their use in urban areas is expected to expand further in the future, especially in logistics and infrastructure inspections.
What is needed in this context is a control system that can properly manage the large number of unmanned aircraft. This is called an unmanned aircraft air traffic management system (UTM).

Although many simulation studies have been conducted on UTMs, few demonstrations have been conducted on actual aircraft. This has not been done and there is little knowledge of it. Therefore, we aim to verify various navigation methods with the actual aircraft to achieve more successful implementation.

We are using PX4, an OSS drone software, as a base for the aircraft modification. In addition, we are using a companion computer approach to control the airframe through the Internet, and a microcontroller such as Raspberry pi for processing other than the main attitude control.

無⼈機交通管理に適したドローンの四次元航法


現在ドローンは技術の発展により我々の生活に急速に普及しおり、今後都市圏における更なる利用拡大が予測され、特に物流やインフラ点検での活用が期待されている。 ここで必要となってくるのが、多くの無人航空機を適切に管理する管制システムである。これを無人航空機航空交通管理システム（UTM）と呼ぶ。

すでに運用されている、旅客機などの有人航空機の運航管理（ATM）とは性質が異なり、より高密度での飛行、目視外範囲での完全自動航行時の衝突回避システムなどが求められる。 UTMに関するシミュレーションの研究は多く行われているが、実際の機体を用いた実証実験はほとんど行われておらず知見が少ない。そのため様々な航法を実際の機体で検証し、より実装に向けた成果を上げることを目指している。

機体の改造は、OSSドローンソフトウェアであるPX4をベースに使用している。また、インターネットを介しての機体制御やメイン姿勢制御以外の処理をさせるためにRaspberry piなどのマイコンを相乗りさせる（コンパニオンコンピュータ）アプローチも行っている。
